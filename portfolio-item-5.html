Reserverings-
systeem

Website voor kappersafspraken

Samen met een klasgenoot ben ik voor een barber aan de slag gegaan om een reserveringssysteem te ontwikkelen. De website bevat zowel front-end als back-end. Klanten kunnen de website bezoeken om informatie over de barber te krijgen, het portfolio van de barber te kunnen inzien, en natuurlijk om een afspraak te maken.

Voor de barber hebben we een afgeschermd admin-gedeelte ontwikkeld waar alle afspraken terug te vinden zijn. Ook kunnen de afspraken aangepast en verwijderd worden. De barber kan ook zelf handmatig afspraken toevoegen, super handig!

Ik ben verantwoordelijk geweest voor het back-end gedeelte, en heb dus alle PHP van dit project geschreven en de database beheerd. Ook heb ik een gebruikershandleiding gemaakt, zowel voor de barber als voor de klanten. Bekijk de code op mijn Github via de button hieronder!
