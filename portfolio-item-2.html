Laravel project

Pokémon Collection

Voor een van de programmeervakken hebben we het PHP-framework Laravel behandeld. Ik heb gekozen om een applicatie te ontwikkelen waar gebruikers op bij kunnen houden welke Pokémon ze al verzameld hebben.

Uiteraard heb ik deze webpagina ontwikkeld met PHP en Laravel. Op de pagina wordt er onderscheid gemaakt tussen admins en gebruikers. Zo hebben admins de rechten om nieuwe Pokémon toe te voegen, bestaande Pokémon te editen en te verwijderen. Elke unieke gebruiker kan een eigen lijst met Pokémon bijhouden door gebruik van een koppeltabel.

Ook kan er gefilterd worden op Pokémon, zowel op type als op naam. Zodra er 2 Pokémon gevangen zijn door een gebruiker kan deze op zijn of haar profielpagina komen en de gegevens daar aanpassen. Bekijk de code op mijn Github via de button hieronder!
